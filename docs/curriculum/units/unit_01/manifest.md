# Unit 1 Manifest: Balance by Design

## Unit Overview
- **ID:** U01
- **Title:** Balance by Design
- **Description:** Introduction to the accounting equation (A=L+E) and its application in creating a Balance Snapshot for a business.
- **Narrative Context:** Sarah Chen needs to organize her startup's initial financial data to understand what she owns, what she owes, and what she's truly worth.

## Lesson Sequence

| Lesson ID | Type | Title | Description |
|-----------|------|-------|-------------|
| U01L01 | launch | Unit Launch: Why Balance Matters | introduce A=L+E and why balance matters using real-world examples. |
| U01L02 | accounting | Classifying the Business | Classify accounts into Assets, Liabilities, and Equity using real business examples. |
| U01L03 | accounting | Business Events & The Equation | Apply A=L+E to simple business events to see the dual-impact idea. |
| U01L04 | accounting | Building the Balance Sheet | Build a simple Balance Sheet structure from categorized data. |
| U01L05 | excel | Spreadsheet Red Flags | Detect and fix common ledger errors using Conditional Formatting. |
| U01L06 | excel | Data Quality & Validation | Enforce data quality with Data Validation lists to ensure consistency. |
| U01L07 | excel | The Balance Snapshot | Compile a clean Balance Snapshot with basic bar chart visuals. |
| U01L08 | project | Sprint 1: Refine & Document | Project work: refine the ledger and create supporting documentation. |
| U01L09 | project | Sprint 2: Visuals & Narrative | Project work: polish the snapshot visuals and craft the narrative explanation. |
| U01L10 | project | Sprint 3: Finalize & Rehearse | Project work: rehearse the micro-pitch and finalize the public product. |
| U01L11 | assessment | Unit 1 Mastery Check | Individual assessment: classify accounts and balance a mini-case. |
