# Capstone Rubrics Inline Content

## Overview

The Capstone rubrics page (`/capstone/rubrics`) currently shows only "Download from overview" placeholders instead of actual rubric content. This track adds inline rubric categories and criteria so students and teachers can preview grading standards without downloading PDFs.

## Functional Requirements

1. Display pitch rubric categories inline with point allocations
2. Display model tour checklist sections inline
3. Maintain PDF download links for detailed reference
4. Match existing page styling and structure

## Non-Functional Requirements

- Keep page consistent with existing capstone page design language
- Ensure mobile-responsive layout
- Maintain accessibility (proper heading hierarchy, readable contrast)
