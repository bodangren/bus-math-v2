# Track Specification: Remaining Exercise Placeholders

## Overview
Implement the 3 remaining exercise component placeholders:
1. profit-calculator
2. budget-worksheet
3. error-checking-system

## Functional Requirements
1. For each placeholder:
   - Create React component following established exercise patterns
   - Add practice.v1 envelope submission
   - Add submittedRef double-submit guard
   - Add reset function that clears submittedRef
   - Use optional chaining on callbacks (onSubmit?.(), onComplete?.())
   - Register in lib/activities/registry.ts
2. Create test files for each component
3. Verify all components follow existing patterns from other exercise components

## Non-Functional Requirements
- All lint and type checks must pass
- All tests must pass
- Build must pass cleanly

## Acceptance Criteria
1. All 3 placeholders are replaced with real components in activity registry
2. Each component has a corresponding test file
3. npm run lint passes
4. npm test passes
5. npm run build passes
