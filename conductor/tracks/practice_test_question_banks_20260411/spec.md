# Practice Test Question Banks Expansion

## Overview
Expand the practice test question banks for Units 2-8 from 1 placeholder question each to full banks matching Unit 1's depth (3+ questions per unit, covering key lessons and learning objectives).

## Functional Requirements
- Each unit (2-8) must have at least 3 practice test questions
- Questions must cover the key lessons defined in each unit's lesson list
- Each question must include: unique id, lessonId, lessonTitle, prompt, correctAnswer, distractors (3), explanation, and objectiveTags
- Questions must be pedagogically sound and aligned with the unit's learning objectives

## Non-Functional Requirements
- Maintain the same question structure and format as Unit 1's existing questions
- Ensure questions are clear, concise, and appropriate for the target audience
- Avoid duplicate or overly similar questions within the same unit

- [ ] Unit 2 has at least 3 practice test questions

- [ ] Unit 4 has at least 3 practice test questions
- [ ] Unit 5 has at least 3 practice test questions
- [ ] Unit 6 has at least 3 practice test questions
- [ ] Unit 7 has at least 3 practice test questions
- [ ] Unit 8 has at least 3 practice test questions
- [ ] All questions follow the established PracticeTestQuestion interface
- [ ] All tests pass
